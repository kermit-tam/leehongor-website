# 圖畫配對遊戲圖片

## 圖片規格

- **格式**: PNG（推薦透明背景）
- **尺寸**: 建議 300x300 像素
- **風格**: 簡單卡通、易於小朋友辨識

## 圖片清單

將圖片放在此文件夾，並按照以下命名：

### 第一課
| 文件名 | 對應生字 | 建議內容 |

## 使用方式

1. 將圖片放入此文件夾
2. 確保文件名與 `chinese-lesson-XX.ts` 中的 `image` 欄位對應
3. 如果沒有圖片，系統會自動使用 emoji 作為後備

## 圖片來源建議

- 自己繪製（推薦，統一風格）
- [Flaticon](https://www.flaticon.com/) - 免費圖標
- [Freepik](https://www.freepik.com/) - 免費插圖
- [Openclipart](https://openclipart.org/) - 公有領域剪貼畫
