## 自動部署 (GitHub Actions)

✅ **已配置** - 每次 push 到 main 分支自動部署

### 使用方法
```bash
git add .
git commit -m "更新內容"
git push origin main
# 自動部署到 Firebase Hosting
```

### 設置 Secrets（只需一次）
1. 到 GitHub Repository → Settings → Secrets → Actions
2. 添加以下 Secrets：
   - `GCP_SA_KEY` - Firebase Service Account JSON
   - `NEXT_PUBLIC_FIREBASE_*` - Firebase 配置

詳情見 `DEPLOY_SETUP.md`
